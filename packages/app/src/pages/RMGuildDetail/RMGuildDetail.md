# RMGuildDetail 仕様書

- 対応ファイル: `src/pages/RMGuildDetail/RMGuildDetail.vue`
- ルート: `/RMGuildDetail/:guildId`
- レイアウト: `layoutAfterLoginNavigator.vue`
- ステータス: 実装済み

## 画面概要
ギルドの詳細情報と所属メンバー一覧を表示するページです。通常は閲覧画面として使い、編集モードを ON にするとメンバーごとのスキル投稿画面へ遷移できる導線が有効になります。

## 主な表示要素
- 読み込み中スピナー
- エラーメッセージ表示領域
- ギルド基本情報カード
  - ギルド名
  - ギルド ID
  - 説明 / メモ
  - 存続状況
  - 設立日
  - ゲーム内 ID
  - 正式メンバー数
- メンバー一覧
- 戻る / ギルド編集ボタン
- 編集モード切替トグル

## 主な操作
- ルートパラメータ `guildId` を使ってギルド情報を取得する
- 編集モード OFF では一覧閲覧のみ
- 編集モード ON ではメンバー行押下で `RMSkillPost` へ遷移する
- 「ギルド編集」で `RMGuildEdit` へ遷移する
- 「戻る」でブラウザバックする

## 利用データ・依存
- `dbGuildModule` によるギルド詳細取得
- Quasar の `QCard`、`QList`、`QToggle` など
- 独自ボタン `RMButton`

## 備考
- メンバー追加・削除 UI はありません。
- 設立日は未設定の可能性があります。
- 幅は最大 800px に制御され、PC では中央寄せ表示です。
