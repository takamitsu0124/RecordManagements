# RMSkillPost 仕様書

- 対応ファイル: `src/pages/RMSkillPost/RMSkillPost.vue`
- ルート: `/guild/:guildId/user/:userId/post-skill`
- レイアウト: `layoutAfterLoginNavigator.vue`
- ステータス: 実装済み

## 画面概要
ギルドメンバーごとのスキル記録と熟練度を管理するページです。多数の画像アップロード、ドラッグ並び替え、カルーセル閲覧を備えた、本アプリ内でもっとも機能量の多い編集画面です。

## 主な表示要素
- ユーザー名付きヘッダー
- 編集モード切替
- 熟練度入力欄（8 系統）
- スキル記録セクション（17 種別）
  - 一覧表示
  - 画像アップロード
  - 削除ボタン
  - ドラッグ並び替え
- フルスクリーン画像カルーセル
- 保存 / キャンセルボタン

## 主な操作
- `guildId` / `userId` を使って対象ユーザー情報を取得する
- 閲覧モードでは画像一覧の確認とカルーセル表示を行う
- 編集モードでは画像追加、削除、順序変更を行う
- 保存時に熟練度を補正し、DB 更新と不要画像削除を実施する
- キャンセル時は前画面へ戻る

## 利用データ・依存
- `dbUserModule`
- Firebase Storage へのアップロード / 削除
- `vuedraggable`
- Quasar の `QUploader`、`QDialog`、`QCarousel` 系コンポーネント
- `RMButton`

## 備考
- 画像種別キーとデータ型の整合が重要です。
- 画像枚数上限や入力補助は強くありません。
- モバイルでの操作体験がプロダクト品質に大きく影響する中心画面です。
