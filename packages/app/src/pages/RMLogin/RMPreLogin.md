# RMPreLogin 仕様書

- 対応ファイル: `src/pages/RMLogin/RMPreLogin.vue`
- ルート: `/RMPreLogin`
- レイアウト: `layoutBeforeLoginNavigator.vue`
- ステータス: 実装済み

## 画面概要
アプリ起動直後のプレログイン画面です。ログイン方法の選択と新規登録への導線を提供します。

## 主な表示要素
- ロゴ
- アプリ名表示
- メールログインボタン
- 電話番号ログインボタン（非表示）
- 新規登録ボタン
- 背景画像

## 主な操作
- メールログイン画面へ遷移する
- 電話番号ログイン画面へ遷移する準備があるが、現在は UI 非表示
- 新規登録画面へ遷移する

## 利用データ・依存
- `RMButton`
- Firebase Storage 上の背景画像 URL

## 備考
- 画面内容はほぼ固定で、動的データ依存はありません。
- 背景画像 URL がハードコードされています。
