# RMUserEdit 仕様書

- 対応ファイル: `src/pages/RMUserEdit/RMUserEdit.vue`
- ルート: `/user/:userId/edit`
- レイアウト: `layoutAfterLoginNavigator.vue`
- ステータス: 実装済み

## 画面概要
ユーザーのプロフィール情報を編集するページです。名前、所属情報、プレイヤー状態、各種日付、連絡先などをまとめて更新できます。

## 主な表示要素
- 読み込みスピナー
- エラーメッセージ表示
- 基本情報入力
  - キャラ名
  - キャラ名かな
  - ギルド ID
  - 所属番号
  - プレイヤー状態
- 日付入力
  - 所属日
  - ゲーム開始日
  - 生年月日
- 連絡先入力
  - メール
  - 電話番号
- キャンセル / 保存ボタン

## 主な操作
- `userId` をもとにユーザー情報を取得してフォームへ反映する
- 日付文字列と `Date` を相互変換する
- 保存時に `dbUserModule.merge()` で更新する
- 完了後はトースト通知して前画面へ戻る

## 利用データ・依存
- `dbUserModule`
- Quasar の `QDate` を含むフォーム UI
- `RMButton`

## 備考
- アイコンアップロード用の下準備はありますが、画面 UI には出ていません。
- 日付フォーマットは `YYYY/MM/DD` ベースです。
